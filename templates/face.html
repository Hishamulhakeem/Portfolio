HumanVision - Face Blurring Tool using MediaPipe

This project detects faces in images1, videos, or webcam streams and applies a blur to each detected face using MediaPipe and OpenCV.

  • Face detection using MediaPipe
  • Blur applied to detected faces
  • Supports:
    • images1 (--mode image)
    • Videos (--mode video)
    • Real-time webcam (--mode webcam)
  • Implemented face blurring using Google's MediaPipe framework, leveraging real-time face detection to accurately locate facial landmarks. The system applies a blur mask over detected regions, ensuring privacy without affecting the rest of the image or video.

Key Features

  • Face Detection: Powered by Google MediaPipe framework for accurate and fast face detection.
  • Privacy Protection: Applies blur effect to detected faces while preserving image quality.
  • Real-time Processing: Works with live webcam feeds for instant face blurring.
  • Multi-format Support: Processes images1, videos, and webcam streams.
  • Easy to Use: Simple command-line interface with different modes.
  • Efficient: Lightweight and fast processing using OpenCV and MediaPipe.
